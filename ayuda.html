QUE ES DREAM-J ?

Dream-J es una versión portable de la serie power Linux esta versión “semi” pequeña se caracteriza por estar mas orientada al uso portable  bajo una maquina virtual “qemu”  ya no tendras pretextos para reiniciar el ordenador  

algunas caracteristicas:

-esta basada en DSL  www.damnsmalllinux.org

-manejo mas orientado a consola

-bajo uso de recursos

-velocidad

-portabilidad

-cabe en un flash memory de 1GB

-usa qemu parcheado para q trabaje con winpcap

A QUE ESTA ORIENTADA DREAM-J ?

Dream-J esta orientada al pentesting o seguridad  también a desarrolladores, programadores, curiosos y publico en gral ya que su manejo
esta mas al uso de la consola  y por esta razon no se le quito el kernel 2.4.31 ni JWM como gestor de ventanas  para reducir  el tamaño de la distro

POR QUE DREAM-J USA EL KERNEL 2.4.31 Y NO UNO DE  LA SERIE 2.6?

Dream-J usa el kernel 2.4.31 por cuestiones de rendimiento velocidad y tamaño esta versión esta diseñada para el uso en una maquina virtual para asi poder darle a dream-j mayor portabilidad bajo cualquier computadora

PUEDO INSTALAR DREAM-J EN MI DISCO DURO?

Si claro es posible pero no lo recomiendo ya que el kernel 2.4.31 fue diseñada para pc´s de bajo rendimiento y es posible q el hardware no sea muy bien reconocido por Dream-j y tengas algunos problemas de todas formas necesitas min 3 GB ve al menu "DSL/Apps>tools>install to hardrive"
o visita:  www.damnsmalllinux.org  para mas informacion y soporte      

Y QUE HAY DEL WIFI?

Por el momento Dream-J no tiene un gran soporte para varios chipsets exepto para atheros,Orinoco,atmel,prism2,Hermes en versiones nuevas de Dream-J se solucionara este problema

COMO CONECTARSE CON WINDOWS?

Dream-J biene con 2 programas que te ayuadaran "Samba" y "SMBClient"  para encontrarlos ve al menu "DSL/Apps>net>"

USO CON QEMU

si estas usando la version  usb con el virtualizador qemu con el script "dreamj.bat"  la conexion a internet se hara de manera automatica no necesitas 
configurar  ni agregar alguna targeta de red  el direccionamiento se hara por dhcp para mas detalles de los scripts revisa el archivo leeme.txt

QEMU PCAP

al usar D.J  con el script "dreamj-pcap.bat" estaras usando a D.J  con  qemu parcheado para q trabaje con winpcap para ver los parametros
que debes usar revisa el archivo leeme.txt
